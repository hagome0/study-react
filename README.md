## 개발환경 구축

1. npm 설치
https://nodejs.org/en/

2. $ npm install -g create-react-app@2.1.8

3. $ mkdir react-app

4. $ cd react-app

5. $ create-react-app . (현재 디렉토리에 react app 생성)

6. $ npm run start (실행이 되는지 테스트)
